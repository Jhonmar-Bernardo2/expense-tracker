# Components

App components live in `resources/js/components/*`.

Conventions:
- Put reusable app-level components under:
  - `navigation/` (sidebar/header/breadcrumbs)
  - `forms/` (form sections, field wrappers)
  - `tables/` (table wrappers, empty states)
  - `charts/` (report charts)
  - `shared/` (confirm dialogs, loading/empty states)

UI kit components live in `resources/js/components/ui/*` (shadcn-vue).
